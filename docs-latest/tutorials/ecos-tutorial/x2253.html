Cirrus Logic ARM CL-PS7111 Evaluation Board Hardware Setup

The implementation of the port of eCos to the Cirrus Logic ARM CL-PS7111 Evaluation Board (also known as EB7111) is based on the EDB7211 port.

For that reason, the setup required is identical to the EDB7211 as described above, with the following exceptions:

  • The Cygmon ROM monitor is not supported

  • The ARM Multi-ICE is not supported

  • Prebuilt GDB stubs are provided in the directory loaders\arm-eb7111 relative to the root of your GDB installation

  • If rebuilding the GDB stub ROM, in step 3 the STUBS_config script should be invoked as (all on one line): 
    sh <BASE_DIR>/packages/hal/arm/cl7211/v1_2_9/misc/STUBS_config <BASEDIR>

  • All remote serial communication is done with the serial I/O connector labelled "Serial Port 1"
