These release notes accompany version 1.2.9 of eCos.

This release of eCos supports the following targets:

  • Matsushita MN10300 (AM31)

  • Matsushita AM33

  • Toshiba TX39 (MIPS R3900 derivative)

  • Motorola PowerPC MPC823, MPC850, and MPC860

  • Fujitsu SPARClite MB86831, MB86832, and MB86833

  • Advanced RISC Machines ARM7 and ARM9 (including Thumb support on cores on which it is implemented)

  • Intel StrongARM

  • NEC VR4300

  • Hitachi SH3

  • Linux i386—synthetic Linux target

This release of eCos supports the following target platforms:

  • Matsushita MN10300 stdevall (AM31)

  • Matsushita STB System Reference Board (AM33)

  • Toshiba JMR3904 (TX39)

  • Cogent CMA 101/102 evaluation boards with a CMA287-23 (MPC823), CMA287-50 (MPC850), or CMA 286-60 daughterboard

  • Motorola MBX860

  • Motorola FADS (MPC860) This board is not supported by Cygnus. See the file hal/powerpc/FADS/v1_2_9/README for details.

  • Fujitsu SPARClite Evaluation Board (SPARClite MB86831, MB86832, and MB86833)

  • ARM PID (ARM7/ARM7t/ARM9)

  • ARM AEB-1 (rev B and C)

  • Cirrus Logic EB7111 evaluation board fitted with CL-PS7111 CPU (ARM710c core)

  • Cirrus Logic EDB7211 evaluation board (aka EB7111-2) fitted with Cirrus Logic EP7211 CPU (ARM7tdmi core)

  • Cogent CMA 101/102 evaluation boards with a CMA230 (ARM7tdmi) daughterboard —support for this platform is still “beta”

  • StrongARM EBSA-285 evaluation board

  • NEC DDB-VRC4373 (VR4300)

  • Hitachi EDK7708

  • Linux (i386) - synthetic Linix target —support for this platform is “beta”

This release of eCos supports the following host operating systems:

  • MicrosoftTM Windows NTTM, Windows 95TM and Windows 98TM—support for Windows 95 and Windows 98 is still “beta”.

  • UNIXTM—support for UNIX is still “beta”, with the exception of the SPARCLite release of eCos where Solaris2.5.1 or later is a fully supported host. Redhat LinuxTM and SolarisTM are the only tested UNIX variants.
