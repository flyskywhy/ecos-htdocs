Tests and examples

Test suites are included for every portion of eCos shipped in this release; these are brief programs that test the behavior of most system calls and libraries in eCos.

Chapter 6 describes how to build and run these test suites. More detail is given in Chapter 13.

These chapters give examples that guide you through running eCos applications, starting from a “Hello world” program and then moving on to more complex programs that use additional kernel features. These are described in Part III in Getting Started with eCosTM.
