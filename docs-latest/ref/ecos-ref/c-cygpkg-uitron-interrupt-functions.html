Component: Interrupt-safe functions

Macro name

CYGPKG_UITRON_INTERRUPT_FUNCTIONS

Type

Dummy

Description

The uITRON system provides some functions which may safely be used within interrupt handlers. In eCos, this means within ISRs, providing that the corresponding DSR is associated with that interrupt. These functions are typically named ixxx_yyy(), according to the uITRON specification, for example isig_sem() corresponds to normal function sig_sem().

Option: Execute in ISR if safe

Macro name

CYGSEM_UITRON_ISRFUNCS_TRY_IMMEDIATE_EXECUTION

Type

Boolean

Description

These functions of necessity maintain a queue of operations requested for deferred execution. However, during an interrupt, it may be safe to perform scheduling operations. If this option is set, the interrupt-safe functions will have effect there and then if it is indeed safe, rather than queueing a request to perform the operation in the DSR.

Option: Deferred operation queue size

Macro name

CYGNUM_UITRON_ISR_ACTION_QUEUESIZE

Type

Enum

Description

These functions of necessity maintain a queue of operations requested for deferred execution. This option controls the queue size. It must be a power of two for implementation reasons.
