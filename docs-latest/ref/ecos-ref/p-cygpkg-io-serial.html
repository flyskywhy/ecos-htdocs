Package: Serial device drivers

Macro name

CYGPKG_IO_SERIAL

Type

Boolean

Description

This option enables drivers for basic I/O services on serial devices.

Component: TTY-mode serial device drivers

Macro name

CYGPKG_IO_SERIAL_TTY

Type

Boolean

Description

This option enables the terminal-like device driver used for serial devices that interact with humans, such as a system console.

Option: Console device name

Macro name

CYGDAT_IO_SERIAL_TTY_CONSOLE

Type

String

Description

This option selects the TTY device to use for the console.

Component: HAL/diag serial device driver

Macro name

CYGPKG_IO_SERIAL_HALDIAG

Type

Boolean

Description

This option enables the use of the HAL diagnostic channel via the standard I/O drivers.

Component: TTY mode HAL/diag channel

Macro name

CYGPKG_IO_SERIAL_TTY_TTYDIAG

Type

Boolean

Description

This option causes '/dev/ttydiag' to be included in the standard drivers.

Component: TTY mode channel #0

Macro name

CYGPKG_IO_SERIAL_TTY_TTY0

Type

Boolean

Description

This option causes '/dev/tty0' to be included in the standard drivers.

Option: TTY mode channel #0 device

Macro name

CYGDAT_IO_SERIAL_TTY_TTY0_DEV

Type

String

Description

This option selects the physical device to use for '/dev/tty0'.

Component: TTY mode channel #1

Macro name

CYGPKG_IO_SERIAL_TTY_TTY1

Type

Boolean

Description

This option causes '/dev/tty1' to be included in the standard drivers.

Option: TTY mode channel #1 device

Macro name

CYGDAT_IO_SERIAL_TTY_TTY1_DEV

Type

String

Description

This option selects the physical device to use for '/dev/tty1'.

Component: TTY mode channel #2

Macro name

CYGPKG_IO_SERIAL_TTY_TTY2

Boolean

Description

This option causes '/dev/tty2' to be included in the standard drivers.

Option: TTY mode channel #2 device

Macro name

CYGDAT_IO_SERIAL_TTY_TTY2_DEV

Type

String

This option selects the physical device to use for '/dev/tty2'.
