Persistence

The Configuration Tool records user interface characteristics (such as window size) in the registry, and these characteristics are automatically reloaded each time the tool is invoked. The location for such items is in the registry key:

HKEY_CURRENT_USER\Software\Cygnus Solutions\eCos\Configuration Tool

The location of the component repository and the most recently accessed build trees are also persistent across Configuration Tool sessions (see the section called The Repository).
