View Settings

Select View->Settings menu item to change aspects of the visual presentation.

The first group of controls allows the specification of a header file viewer. This is the application that will be launched by the Configuration Tool if you double-click a filename in the Properties Table or header files dialog.

The second group allows the specification of an HTML viewer. You can elect to use the internal browser control only if Internet Explorer version 3 or later is installed.

The next options control the visual presentation of the Configuration Pane:

  • Display macro names

    Check this option if you wish to see the tree control displayed using macro names as opposed to the normally displayed descriptions. This option also affects the display of configurable items in various other contexts throughout the Configuration Tool.

  • In-cell edit controls for binary items

    Uncheck this option if you do not wish to see in-cell representation of Boolean and Radio configurable items (the value of these items may be changed using the checkbox in the tree control).

  • Automatically expand and collapse tree

    Uncheck this option if you do not wish Boolean and Radio tree nodes to expand when the ancestor item is set and to collapse when the ancestor item is unset.

  • Display integer values in hexadecimal format.

    Check this item if you wish to see the tree control display integer items in hexadecimal format. The option also affects the display of configurable items in various other contexts throughout the Configuration Tool. Note that the output of integer items to header files is not affected (items are always output in decimal).

The options in the Font groupbox allow you to customize the fonts for the individual panes of the Configuration Tool. To change a font, select the desired pane using the combo box, and then click the Change Font... button. A font selection dialog will appear, allowing you to select the desired font.
