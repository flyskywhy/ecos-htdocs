Rules Help

The Rules window provides a display of consistency rules and their state. The window can be made visible by selecting View-> Rules:

Show Failing Rules

Only those consistency rules that are currently failing will be displayed.

Show All Rules

All rules will be displayed.

Hide

The rules window will be hidden.

Consistency rules are displayed in three columns. The left and right columns indicate the configuration items involved, and the center column indicates the relationship that the rule enforces. Also in the left column is an indicator icon: a check to indicate a rule that is satisfied or an exclamation point to indicate a rule that fails. The icons for rules that are inactive (whose containing items are disabled) will be grayed.

You can navigate to a rule"s containing item by double-clicking on an entry in the Rules window. This action causes the appropriate item in the Configuration Pane to be selected.

The Always on Top item in the Rules window system menu allows the Rules window to be made a topmost window.
