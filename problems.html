Reporting Problems

Reviewing problems

If you believe you have found a problem with eCos sourceware, check the known problems list. The problem may have been reported already. This list contains suggestions for working around certain known problems.

Submitting a problem report

If you have been unable to find reference to the problem you are experiencing in the known problems list, please submit a problem report using the eCos sourceware problem report form. It is important to provide as much information as possible to enable our engineers to reproduce and analyse the problem. By providing a problem report, you will assist us in improving the software for the benefit of all users.

Please note that we are unable to provide technical support to eCos sourceware users. Cygnus can provide mission-critical support to those who require it. Please see our corporate web site for further details.
