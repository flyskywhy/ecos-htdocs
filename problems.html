Reporting Problems

Reviewing problems

If you believe you have found a problem with eCos sourceware, check the eCos problem database. The problem may have been reported already. The database contains suggestions for working around certain known problems.

Submitting a problem report

If you have been unable to find reference to the problem you are experiencing in the problem database, please submit a problem report using the eCos sourceware problem report form. It is important to provide as much information as possible to enable our engineers to reproduce and solve the problem.
