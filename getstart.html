Downloading and Installation

Host support

The eCos net distribution is available in both Linux and Windows versions. The Linux version has been tested under Red Hat Linux 6.1 (and later releases) but should work under most Linux variants. The Windows version has been tested under Microsoft Windows NT 4.0 and also works under Windows 95, Windows 98 and Windows 2000. However, please note that it is not possible to build the eCos development tools under Windows 95 or Windows 98 at this time.

Installation

Instructions for downloading and installing eCos are available for both Windows and Linux users. Each download includes eCos sources, documentation, host tool sources and appropriate host tool binaries. Click on one of the following links for full details:
  • Installing eCos under Linux
  • Installing eCos under Windows
The eCos TCP/IP stack beta release is available as a separate download in the eCos package distribution format:
  • eCos TCP/IP stack 1.0b1 (net-1.0b1.epk - 515kB)
Refer to the Managing the Package Repository section of the eCos User's Guide for details of installing the TCP/IP stack using the eCos Package Administration Tool.

Experienced developers may wish access the very latest eCos repository using our anonymous CVS server.

Documentation

Full documentation is available online in both HTML and PDF formats. It is included within the eCos net distribution in HTML format. The HTML documentation may also be downloaded separately in a selection of archive formats:
  • eCos 1.3.1 documentation (eCosDoc131.zip - 2.7MB)
  • eCos 1.3.1 documentation (ecos-1.3.1-doc.tar.bz2 - 2.3MB)
  • eCos 1.3.1 documentation (ecos-1.3.1-doc.tar.gz - 2.7MB)
New users are encouraged to follow the programming tutorial in the appropriate Getting Started with eCos document which guides the reader through the stages involved in building and running an eCos application.

Preliminary TCP/IP stack documentation is also available for browsing.

Previous releases

Previous public releases of eCos remain available for downloading. Users of eCos 1.2.1 may refer to the eCos 1.2.1 installation instructions for Linux and Windows.
