eCos Sourceware

Getting Started

Downloading

The eCos sourceware distribution is available in both Linux and Windows versions. The Linux version has been tested under RedHat/Linux 5.1 but should work under most Linux variants. The Windows version has been tested under Windows NT 4.0 and also works under Windows 95 and Windows 98. Click on one of the following links to download:
  • eCos 1.1 sourceware for Linux (4MB tar.bz2 file)
  • eCos 1.1 sourceware for Windows (10MB self-extracting executable)
In order to build the eCos sources, you will require a cross-compiler and related development tools. The necessary tools are available in source form only. Click on the following link to download:
  • eCos sourceware development tools
The eCos sourceware development tools are designed to be built using GNU native tools on either Linux or Windows NT. Linux users will already have the necessary tools. Windows NT users should use pre-built GNU-Win32 native tools. Click on the following link to download from one the GNU-Win32 mirror sites:
  • GNU-Win32 development tools
Note that, although eCos may be configured and built on Windows 95 and Windows 98, it is not possible to build the eCos sourceware development tools on these platforms. If you wish to develop eCos applications using Windows 95 or Windows 98, you must build the development tools under Windows NT.
