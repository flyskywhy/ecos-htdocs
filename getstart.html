Downloading and Installation

Host support

The eCos net distribution is available in both Linux and Windows versions. The Linux version is tested under recent versions of the Red Hat, SuSE and Debian distributions for x86 and should work under most Linux variants. The Windows version has been tested under Microsoft Windows 2000 Professional and should also work under Windows 95, Windows 98, Windows ME and Windows XP. However, building the eCos GNU cross toolchains under Windows 95/98/ME has been found to be unreliable. We recommend that users avoid building the development tools on these systems if possible.

Installation

Cygwin

Developers wishing to install eCos on a Windows host must first install a recent version of the Cygwin UNIX emulation system. Full instructions on installing Cygwin for use with eCos are available. The following instructions assume that Cygwin has already been installed (where necessary) and that the reader is familiar with invoking a bash shell.

Toolchain

The eCos 2.0 beta release features an installation tool which simplifies the downloading and installation of the eCos sources, host tools and documentation. The installation tool can optionally download one or more pre-built GNU cross toolchains (contributed by eCosCentric Limited) for use in conjunction with eCos. At present, toolchains for the following target architectures are available in pre-built form:

ARM (including StrongARM, XScale)arm-elf
Hitachi SuperHsh-elf
Intel x86i386-elf

Developers targetting one of the other architectures must build a toolchain themselves at present. Full instructions for downloading source code and building a toolchain are available.

eCos

We recommend that eCos is installed to /opt/ecos where it will be accessible by all users. This may require installation by a user with suitable privileges. First, download the eCos installation tool by using the following command at a bash prompt:

    wget --passive-ftp ftp://sources.redhat.com/pub/ecos/ecos-install.tcl

The installation tool may then be invoked by using a command of the form:

    sh ecos-install.tcl INSTALLPATH [ TARGET ... ]

For example, to download and install eCos with both the ARM and Intel x86 pre-built toolchains to the /opt/ecos directory, use the following command:

    sh ecos-install.tcl /opt/ecos arm-elf i386-elf

The installation tool will present a list of mirror sites from which eCos may be downloaded. For best results, please select a mirror site in your own geographical region.

Windows users should note that POSIX-style paths are relative to the root of their Cygwin installation (typically c:\cygwin) by default. For example, /opt/ecos might be located at c:\cygwin\opt\ecos in the Windows Explorer.

Following installation, individual users of eCos should extract the default platform information which the eCos Configuration Tool uses to run tests. For Windows users, this can be achieved by double-clicking on the platforms.reg file in the Windows Explorer. This file may be found in the ecos-2.0b1/tools/bin directory under the location at which eCos was installed. Linux users, should execute the following command from the location at which eCos was installed:

    tar -C $HOME -xf ecos-2.0b1/tools/bin/platforms.tar

Users may also wish to create a shortcut to the eCos Configuration Tool on their desktop. Typically, this may be achieved by dragging the configtool or configtool.exe executable file from the file manager provided by your operating system onto the desktop. This file is located in the ecos-2.0b1/tools/bin directory under the location at which eCos was installed.

Documentation

Full documentation is available online in both HTML and PDF formats. It is included within the eCos net distribution in HTML format only. The HTML documentation may also be downloaded separately in multiple archive formats:

  • eCos 2.0 beta documentation (eCosDoc20b1.zip - 2.9MB)
  • eCos 2.0 beta documentation (ecos-2.0b1-doc.tar.bz2 - 1.8MB)

New users are encouraged to follow the programming tutorial in the appropriate Getting Started with eCos document which guides the reader through the stages involved in building and running an eCos application.

Previous releases

Previous public releases of eCos remain available for downloading. Users of eCos 1.3.1 may refer to the eCos 1.3.1 installation instructions for Linux and Windows.
