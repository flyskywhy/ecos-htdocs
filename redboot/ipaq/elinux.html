Embedded Linux demo

Building the Embedded Linux kernel

The sources for the Linux kernel running on the iPAQ can be obtained from the Handhelds project. Follow the directions to download the kernel source from anonymous CVS.

Once you have the sources in place, apply this patch file. The kernel can then be built: 

    cd XXX/linux/kernel
    patch -p0 < elinux.patch
    make zImage
