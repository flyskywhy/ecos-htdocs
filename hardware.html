Supported Hardware

Support for new targets and devices is announced via the ecos-announce mailing list. For further information concerning the porting of eCos, see the porting section of the FAQ.

Note: recently the anonymous CVS repository has been updated with new ports and device drivers in preparation for the eCos v2.0 release. More details can be found in this mailing list post.

Architecture Index

ARM, IA32, Matsushita AM3x, MIPS, NEC V8xx, PowerPC, SPARC, SuperH

Devices Index

Flash devices, Ethernet devices, Serial devices, USB devices, Timekeeping devices
