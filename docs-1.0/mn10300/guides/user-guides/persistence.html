Persistence

The Configuration Tool records user interface characteristics such as window size in the registry. When the Configuration Tool is invoked, such characteristics are automatically reloaded to restore the user interface to the same state in which it was last used. The location for such items is in the registry key

HKEY_CURRENT_USER\Software\Cygnus Solutions\eCos\Configuration Tool

The location of the source repository and the most recently accessed build-trees are also persistent across Configuration Tool sessions (see the section called The Repository).
