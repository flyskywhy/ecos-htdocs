Context-Sensitive Help

You can get context-sensitive help for the Configuration Tool itself:

  • By hitting F1. This will display an HTML page appropriate to the currently focused pane.

  • By clicking the Help buttons in a property sheet dialog box.

You can obtain context-sensitive help relating to a configurable item by hitting Shift+F1 while a configurable item is selected. This will display a floating help window.

The text in the floating window will change with the selection in the Configuration Pane.
