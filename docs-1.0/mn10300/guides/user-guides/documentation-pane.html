Documentation Pane

NOTE: The documentation pane uses the ActiveX control provided by Internet Explorer (IE). The view is only available if IE3 or IE4 is installed — search facilities are available only if IE4 is installed. No version of IE is shipped with the product: if required it may be downloaded from http://www.microsoft.com/ie/

The documentation pane is a vertical pane on the right hand side of the Configuration Tool frame window; it has the full functionality of Internet Explorer: as such it is only described in brief here.

Navigation

You can navigate to the documentation relating to a configurable item by:

  1. Using the pop-up context menu displayed when the item is right-clicked

  2. By double-clicking on the URL item in the properties table

  3. By using the Find dialog and clicking the documentation option.

You can navigate from one documentation page to another in the usual way by clicking HTML links. "Back", "Forward", "Refresh", "Home" and "Stop" operations are available using the toolbar:

Other Operations

You can right-click in the documentation pane (on a link or on body text) to access other functions of Internet Explorer; if you have a mouse wheel you can change the font by holding down the control key and moving the wheel.

You can drag documents such as HTML or RTF files from the desktop explorer to the document pane.
