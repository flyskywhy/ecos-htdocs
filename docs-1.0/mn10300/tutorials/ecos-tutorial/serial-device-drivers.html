Serial device drivers

We provide serial device drivers for the serial ports on your evaluation board. They provide an API (documented in the eCos Reference Manual) to control serial ports directly. The standard I/O library can be configured to use them as a transport layer, and the kernel to use them as a diagnostic channel.
