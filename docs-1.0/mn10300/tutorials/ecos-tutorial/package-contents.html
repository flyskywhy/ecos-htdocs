Chapter 2. Package Contents

This documentation should have arrived in a package that contains the following items:

  • Three books with eCos documentation: Getting started with eCos, the eCos User Guide and the eCos Reference Manual.

  • µITRON 3.0 An Open and Portable Real-Time Operating System for Embedded Systems, a book by Dr. Ken Sakamura.

  • Complete GNUPro Toolkit documentation set (7 books).

  • Printed GNUPro Toolkit architecture specific User's Guide.

  • eCos version 1.0 CD

  • eCos specific monitor PROMs for your evaluation board
