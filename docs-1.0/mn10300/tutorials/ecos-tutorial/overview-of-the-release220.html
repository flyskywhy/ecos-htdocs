GNU tools and their documentation

Cygnus's GNUPro Toolkit, which includes the gcc and g++ compilers and the gdb debugger, is needed to build eCos applications, and it is bundled with this eCos release.

We include the full printed GNUPro documentation, as well as a specific user guide for this hardware architecture. Electronic versions of these documents are also on the CDROM. The full GNUPro documentation can also be found on the web at http://www.cygnus.com/pubs/gnupro/

The version of GNUPro bundled with eCos contains some new features which are not yet in the standard release, including selective linking of functions and data and constructor priority ordering. These features are described in the GNUPro User Guide for this architecture.
