MN10300 installation

The eCos package comes with a pair of EPROMs which provide gdb support for the Matsushita MN10300 series evaluation board via CygMon, the Cygnus ROM monitor. Images of these EPROMs are also provided at loaders\mn10300-stdeval1\cygmon.bin under the root of your eCos installation. The LSB EPROM (LROM) is installed to socket IC8 on the board and the MSB EPROM (UROM) is installed to socket IC9. Attention should be paid to the correct orientation of these EPROMs during installation.

The CygMon stubs allows communication with gdb via the serial port at connector CN2. The communication parameters are fixed at 38400 baud, 8 data bits, no parity bit and 1 stop bit (8-N-1). No flow control is employed. Connection to the host computer should be made via a standard RS232C serial cable (not a null modem cable). The use of a gender changer may also be required.
