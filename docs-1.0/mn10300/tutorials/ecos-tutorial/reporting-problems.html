The email interface for submitting PRs

We have tried to ensure that it is as easy as possible for you to report any problems which may occur while you develop applications using eCos. This is because we wish both to help you with any difficulties and to take every opportunity to improve our software. Cygnus has a reputation for its responsiveness to bug reports in all its products. Maintaining and developing that reputation is an integral part of the eCos project. We will be able to most successfully address the problems you report if you provide as much detail describing the problems as possible.

We would like to thank you in advance for the time you spend supplying us with information about any problems you encounter.

If our Japanese eCos partners need assistance with English language translation or in filing Problem Reports, please contact Nihon Cygnus using the email address, telephone number or postal address you received in the cover letter.

The web interface for submitting PRs

It is possible to submit problem reports (PRs) using the new eCos web problem reporting site located at: http://www.cygnus.com/ecos/ecos-pr.html

The older method of submitting PRs via e-mail is still valid, but we encourage you to try the eCos problem reporting web site.

NOTE: You must include your submitter ID in the form. You will find your company's submitter ID in the cover letter you received with the eCos package.
