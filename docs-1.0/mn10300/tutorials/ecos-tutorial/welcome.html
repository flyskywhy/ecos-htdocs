Chapter 1. Welcome

The Embedded Cygnus Operating System (eCos™) software is a set of tools and a runtime environment for developing embedded real-time applications. eCos is a configurable, open source system that enables the developer to build a runtime system that matches the exact needs of their application. The resulting system needs fewer resources and operates more efficiently. Configuration allows developers to quickly and easily specify their applications requirements from a rich set of runtime features and functionality. The availability of source provides you with complete control and freedom to customize, analyze, and enhance your system.

The eCos development environment is based around an enhanced Cygnus GNUPro compiler toolset, providing the developer with best in class tools to compile, debug and simulate their eCos application.

This documentation has been included in both HTML and printed forms. If you would like to browse the HTML version of these release notes, you can click on the Start menu button in Windows NT, then select the Programs menu, and then the Cygnus eCos menu, and finally select the Documentation item. Your web browser will then display eCos documentation top level HTML page.
