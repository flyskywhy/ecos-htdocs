µITRON

We provide a µITRON compatibility layer, implemented as a configuration of the eCos kernel. This layer implements the complete µITRON 3.02 “Standard functionality” (level S) specification, as well as some of the “Extended” (level E) functions. The µITRON implementation is described in more detail in the eCos Reference Manual.
