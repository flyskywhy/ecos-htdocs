Required

  • Standard PC running Microsoft Windows NT version 4.0 (Intel Architecture). eCos has been tested with both the English and Japanese versions of NT 4.0.

  • Approximately 281 megabytes of free disk space.

  • One (16550 based) serial port on the PC.

  • Matsushita MN10300 standard evaluation board with eCos CygMon Debug PROMs installed.

  • One standard modem (straight connection) serial cable to connect the serial port on the PC to the evaluation board. An optional second serial cable can be used for diagnostic I/O.
