Component: Alarm Handlers

Macro name

CYGPKG_UITRON_ALARMS

Type

Boolean

Description

uITRON Alarm Handlers are used with functions named def_alm() and ref_alm(); they support simple timing, with a function callback at the end of the timed period.

Option: Number of alarm handlers

Macro name

CYGNUM_UITRON_ALARMS

Type

Count

Description

The number of uITRON alarm handlers present in the system. Valid alarm handler numbers will range from 1 to this value.
