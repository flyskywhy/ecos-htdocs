TX39 installation

The eCos package comes with a pair of ROMs which provide gdb support for the Toshiba JMR-TX3904 RISC processor reference board via CygMon, the Cygnus ROM monitor. Images of these ROMs are also provided at loaders\tx39-jmr3904\cygmon.bin under the root of your eCos installation. The ROMs are installed to sockets IC6 and IC7 on the memory daughterboard according to their labels. Attention should be paid to the correct orientation of these ROMs during installation.

The gdb stub allows communication with gdb via the serial port (channel C) at connector PJ1. The communication parameters are fixed at 38400 baud, 8 data bits, no parity bit and 1 stop bit (8-N-1). No handshaking is employed. Connection to the host computer should be made via an RS232C null modem cable.
