Recommended

  • The application of NT 4.0 Service Pack 3 is strongly recommended prior to installation of eCos: most testing of GNUPro, CygWin32 and eCos has been done with Service Pack 3.

  • Pentium II and 64MB or more RAM for best build performance. 55 megabytes of extra disk space if you wish to rebuild the system from the supplied sources.

The system has only been tested in the recommended configuration above, although other configurations are expected to work.
