Tests and examples

We provide test suites for every portion of eCos shipped in this release; these are brief programs which test the behavior of most system calls and libraries in eCos.

In Chapter 15 we describe how you can build and run the test cases we provide. More detail is given in Chapter 6.

We also provide examples which guide you through learning how to write eCos applications, starting from a simple “Hello world” program, but moving on to more interesting programs that use more kernel features. These are described in the Programming Tutorial.
