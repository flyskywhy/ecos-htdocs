The eCos HAL

eCos includes a Hardware Abstraction Layer (HAL) which allows the kernel to be portable to new architectures and platforms. Notes on porting the HAL to new platforms are provided in the eCos Reference Manual, appendix The eCos Hardware Abstraction Layer, section Kernel porting notes.
