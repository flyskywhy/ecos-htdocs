Re-build and configure capability

eCos is distributed with full source code for all runtime components, thus enabling very fine grained configuration of the system.

Cygnus ships a Configuration Tool for eCos, and the eCos packages shipped by Cygnus are configurable using this tool. See Chapter 14 and the eCos User Guide for a demonstration of how to use the Configuration Tool.
