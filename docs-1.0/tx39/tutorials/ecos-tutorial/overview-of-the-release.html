Version 1.0 of eCos ships with the following features:

`Ready to go' system

Pre-compiled binaries for the kernel and the test suites are provided, so that users can immediately verify hardware, run tests, and link to their own applications. These pre-compiled binaries are built with the default eCos configuration.
