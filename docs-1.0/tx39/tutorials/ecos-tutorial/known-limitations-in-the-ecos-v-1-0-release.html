Chapter 8. Known limitations in the eCos version 1.0 release

Known bugs in eCos. There are no known bugs in the current eCos distribution. Please examine the README file at the base of the distribution to see if any problems are discovered after this document goes to print.

Known bugs in GNUPro. Here is the list of known bugs in the GNUPro toolkit as it relates to eCos:

  • The simulators built into gdb have a memory leak when repeatedly loading and running different binaries. After a certain number of loads, there can be a VM allocation failure. Work around: always re-run gdb when you wish to load a different binary.

  • When CygMon is running in command line mode, the load instruction to gdb does not function correctly. Workaround: use gdb instead of CygMon to load executables.

Last minute problems. Any problems that were detected too late to be included in this chapter will be documented in the README file located in the base directory of the eCos installation. Please read this file.
