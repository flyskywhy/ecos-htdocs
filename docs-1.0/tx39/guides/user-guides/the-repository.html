The Repository

When the Configuration Tool is first invoked, it reads type default information for configurable items from a source repository. The location of the source repository is determined (in descending order of precedence) from the following:

  • The previously accessed source repository (see the section called Persistence)

  • The repository as set by the eCos installation procedure

If the repository cannot be located by any other means, you will be required to provide its location when the Configuration Tool is started. This should not occur if the eCos installation procedure has been followed, and will in any case be a one-off interaction.
