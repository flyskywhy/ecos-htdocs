Searching

You can search for text by using the Edit->Find menu item or equivalent toolbar button:

Except in the case of the output pane (see the section called Output Pane in Chapter 2) the normal find dialog controls (match case etc...) are augmented by a set of radio buttons which offers

  • Item Names

    The search will be for an item whose normally displayed name matches the search string.

  • Macro Names

    The search will be for an item whose macro name matches the search string.

  • Documentation

    The search will be for a documentation page that contains the search string. If the string is found, the corresponding page will be displayed in the documentation pane.

    NOTE: this functionality is only available if IE4 is installed.

  • Output

    The search will be in the output pane
