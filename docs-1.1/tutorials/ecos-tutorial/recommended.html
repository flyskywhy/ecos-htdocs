Recommended

  • We strongly recommend that Windows NT users install the NT 4.0 Service Pack 3 before installing eCos: most testing of GNUPro, CygWin32 and eCos has been done with Service Pack 3.

  • We recommend that Windows NT users install Internet Explorer 4.0, since this will allow the Configuration Tool's documentation pane to be searchable.

  • Pentium II and 64MB or more RAM for best build performance.

The system has been tested only in the recommended configuration above, although other configurations are expected to work.
