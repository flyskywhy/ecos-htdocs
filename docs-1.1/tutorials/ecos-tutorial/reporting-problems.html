Chapter 5. Reporting Problems

Reporting bugs and other problems is very important: it allows us to solve your problem quickly, and improves the eCos product. We appreciate the effort you make in reporting problems.

To submit a problem report, please use our web interface. If you have a CD distribution of the eCos Developer's Kit, you should use the address http://www.cygnus.com/ecos/ecos-pr.html and if you are using the Sourceware distribution you should use the address http://www.cygnus.com/ecos/problemreport.html.
