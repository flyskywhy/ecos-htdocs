These release notes accompany version 1.1 of eCos.

This release of eCos supports the following target microprocessors:

  • Matsushita MN10300

  • Toshiba TX39

  • PowerPC - support for the PowerPC is still “beta”

Note: In the current release, eCos is using a subset of the PowerPC architecture. Only the initial (and very simple) memory mapping relies on MPC860-specific features. Everything else relies on features that are common to the entire PowerPC family, which should make ports to other PowerPC CPUs simple.

This release of eCos supports the following host operating systems:

  • MicrosoftTM Windows NTTM, Windows 95TM and Windows 98TM — support for Windows 95 and Windows 98 is still “beta”.

  • UNIXTM (specifically Linux) — support for UNIX is still “beta”. Linux is the only tested UNIX variant.
