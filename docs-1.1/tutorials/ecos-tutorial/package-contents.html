eCos Sourceware release

The eCos Sourceware release consists of the archive files for GNUPro and eCos which which are located on the Sourceware web site http://www.cygnus.com/ecos/.

The Sourceware release does not provide PROMs for the various development boards, but the ROM images for the three supported platforms are in the distribution, so you can burn your own PROMs to work with eCos. HTML versions of the manuals are also in the distribution.
