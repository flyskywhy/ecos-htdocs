How can you use this data?

These measurement data can be used in a number of ways. The most typical use will be to compare different real-time kernel offerings on similar hardware. Another use will be to estimate the cost of implementing a task using eCos. The end user can examine her application and see what effect the kernel operations will have on the total execution time. A final use would be to observe how the tuning of the kernel affects overall operation.
