The Repository

When the Configuration Tool is first invoked, it reads default type information for configurable items from a component repository.

The Configuration Tool will look for the previously accessed component repository (see the section called Persistence), and then it will look for the repository as set by the eCos installation procedure.

If the repository cannot be located, you will be required to provide its location when the Configuration Tool starts.
