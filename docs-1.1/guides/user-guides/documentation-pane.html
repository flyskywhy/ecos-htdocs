Documentation Pane

The documentation pane is located on the right side of the Configuration Tool window; it has the full functionality of Internet Explorer.

Note: The documentation pane uses the ActiveX control provided by Internet Explorer (IE). The view is only available if IE3 or IE4 is installed — the ability to search documentation is available only if IE4 is installed. IE is not shipped with eCos: see http://www.microsoft.com/ie/ to download the latest version.

Navigation

You can navigate to the documentation relating to a configurable item by:

  • using the pop-up context menu displayed when the item is right-clicked

  • double-clicking on the URL item in the properties table

  • using the Find dialog and clicking the documentation option.

Other Operations

You can right-click in the documentation pane (on a link or on body text) to access other functions of Internet Explorer. If you have a mouse wheel (a feature available on new Microsoft mice) you can change the font by holding down the control key and moving the wheel.

You can open documents such as HTML or RTF files by dragging them from the desktop explorer to the document pane.
