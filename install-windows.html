Installing eCos under Windows

Important note

The eCos host tools and GNU development tools are dependent on the forthcoming full release of the Cygwin tools (scheduled 2000-03-29). It will not be possible to build eCos on Windows hosts prior to the release of the new Cygwin tools.

Downloading eCos

The eCos 1.3.1 distribution for Windows is provided as a self-extracting executable file. Click on the following link to download:
  • eCos 1.3.1 for Windows (eCos131.exe - 16MB)
You may prefer to use one of the eCos mirror sites to reduce download time, or if the FTP server is temporarily overloaded.

eCos also requires the GNU native tools for building the development tools and eCos packages. Windows users should obtain the forthcoming full release of the Cygwin tools (scheduled 2000-03-29). Note that the Cygwin tools also include the bzip2 utility required to extract development tool sources.

Installing eCos

  1. Install the Cygwin native tools according to instructions provided on the Cygwin web site.

  2. Install eCos by running the downloaded distribution file and following the on-screen instructions. Note that it is not possible to upgrade an existing eCos installation. Any previous installations should be uninstalled (using the Add/Remove Programs utility in the Control Panel) before installing eCos 1.3.1.

Downloading and installing the development tools

In order to build the eCos sources, you will require a cross-compiler and related development tools. Instructions for downloading and building the tools vary depending on the architecture to be supported. These instructions should be followed from within the bash command-line environment provide by Cygwin. Click on the required toolset for detailed instructions:
  • ARM (arm-elf)
  • ARM in Thumb mode (thumb-elf)
  • Fujitsu SPARClite (sparclite-elf)
  • Hitachi SH (sh-elf)
  • Intel x86 (i386-elf)
  • MN10300/AM33 (mn10300-elf)
  • NEC VR4300 (mips64vr4300-elf)
  • PowerPC (powerpc-eabi)
  • Toshiba TX39 (mips-tx39-elf)
