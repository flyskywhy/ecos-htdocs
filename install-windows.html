Installing eCos under Windows

Downloading

The eCos 1.1 distrubution for Windows is provided as a self-extracting executable file. Click on the following link to download:
  • eCos 1.1 sourceware for Windows (10MB self-extracting exe file)
In order to build the eCos sources, you will require the a cross-compiler and related development tools. The necessary tools are designated ecosSWtools-981021 and are available in source form only. Note that, although eCos may be configured and built under Windows 95 and Windows 98, it is not possible to build the eCos development tools on these platforms. If you wish to develop eCos applications using Windows 95 or Windows 98, you must build the development tools under Windows NT. Click on the following link to download:
  • eCos development tools (33MB zip file)
eCos also requires the GNU native tools for the building of eCos packages. Windows users should obtain the Cygwin Beta 20 version of these native tools. The full binary release is required. Click on the following link to download:
  • Cygwin Beta 20 development tools (12MB self-extracting exe file)

Installing

  1. Install the Cygwin native tools by running the downloaded distribution file and following the on-screen instructions. The remaining instructions assume that the Cygwin tools are installed to the default location. If you wish to install to a different location, you should avoid the use of spaces in the destination path. Once installed, select Cygwin Beta 20 from the start menu hierarchy to launch the bash command-line environment.

  2. Build the eCos development tools (ecosSWtools-981021) by following these steps at the bash prompt:

    • Mount the logical drive(s) you will use to build the eCos development tools:

      mount C: /c

    • Mount the Cygwin tools as /bin such that /bin/sh exists:

      mount C:/cygnus/cygwin-b20/H-i586-cygwin32/bin /bin

    • Mount the Cygwin libraries as /lib:

      mount C:/cygnus/cygwin-b20/H-i586-cygwin32/i586-cygwin32/lib /lib

    • Set the MAKE_MODE environment variable to "unix":

      export MAKE_MODE=unix

    • Extract the eCos development tool sources from the downloaded distribution file to C:\cygnus using a utility such as WinZip. You should ensure that the utility is configured to preserve the directory structure within the zip file during extraction. The directory C:\cygnus\ecosSWtools-981021\src should be generated and populated during the extraction process.

    • Create an object directory for your builds:

      mkdir /c/build
      cd /c/build

    • Configure the tools, saving the output to configure.out:

      /c/cygnus/ecosSWtools-981021/src/configure -v \
        --prefix=/c/cygnus/ecosSWtools-981021 \
        --exec-prefix=/c/cygnus/ecosSWtools-981021/H-i586-cygwin32 \
        --host=i586-cygwin32 --target=target-triplet > configure.out 2>&1

      You should replace target-triplet in the above command with one of the supported target triplets: mn10300-elf, mips-tx39-elf or powerpc-eabi.

    • Build and install the tools, saving the output to make.out:

      make -w all install > make.out 2>&1

    When the build is complete, the executable files of the eCos development tools may be found in the directory C:\cygnus\ecosSWtools-981021\H-i586-cygwin32\bin.

  3. Install eCos by running the downloaded distribution file and following the on-screen instructions.
