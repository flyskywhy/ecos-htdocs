eCos License Overview

The CEPL: an open source license

eCos is licensed under the Cygnus eCos Public License (CEPL), an open source license.

The goal of the CEPL is to guarantee to you that eCos will always be free, and that you will always be free to modify and redistribute eCos.

We consider eCos an essential part of embedded software infrastructure, like GCC and GDB, and think it should be free.

The origin

The CEPL is based on Netscape's Mozilla Public License (MPL), which in turn is inspired by the GNU General Public License (GPL).
