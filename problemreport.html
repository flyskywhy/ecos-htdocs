eCos Problem Reports

Reviewing problems

If you believe you have found a problem with eCos, check the known problems list. The problem may have been reported already. This list contains suggestions for working around certain known problems.

Please note that we are unable to provide technical support to users of the eCos net distribution. Red Hat can provide support via the eCos Developer Support Program for those who require it.

Submitting a problem report

If you have been unable to find reference to the problem you are experiencing in the known problems list, please submit a problem report to Red Hat using the following form. By providing a problem report, you will assist us in improving the software for the benefit of all users. It is important to provide as much information as possible to enable our engineers to reproduce and analyse the problem.
