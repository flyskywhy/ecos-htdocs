Installing eCos under Linux

Downloading

The eCos 1.1 distrubution for Linux is provided as a tar.bz2 archive file. Click on the following link to download:
  • eCos 1.1 sourceware for Linux (4MB tar.bz2 file)
In order to build the eCos sources, you will require the a cross-compiler and related development tools. The necessary tools are designated ecosSWtools-981021 and are available in source form only. Click on the following link to download:
  • eCos development tools (33MB zip file)

Installing

  1. Build the eCos development tools (ecosSWtools-981021) by following these steps at a sh, ksh or bash prompt. Users of the csh and tcsh shells should replace "> filename.out 2>&1" with ">& filename.out" throughout:

    • Extract the eCos development tool sources from the downloaded distribution file to /usr/cygnus using the unzip utility or similar. You should ensure that the utility is configured to preserve the directory structure within the zip file during extraction. The directory /usr/cygnus/ecosSWtools-981021/src should be generated and populated during the extraction process.

    • Create an object directory for your builds:

      mkdir /tmp/build
      cd /tmp/build

    • Configure the tools, saving the output to configure.out:

      /usr/cygnus/ecosSWtools-981021/src/configure -v \
        --prefix=/usr/cygnus/ecosSWtools-981021 \
        --exec-prefix=/usr/cygnus/ecosSWtools-981021/H-host-triplet \
        --target=target-triplet > configure.out 2>&1

      You should replace target-triplet in the above command with one of the supported target triplets: mn10300-elf, mips-tx39-elf or powerpc-eabi. You should replace host-triplet with the host triplet for your build system. You can determine the host triplet by running /usr/cygnus/ecosSWtools-981021/src/config.guess.

    • Build and install the tools, saving the output to make.out:

      make -w all install > make.out 2>&1

    When the build is complete, the executable files of the eCos development tools may be found in the directory /usr/cygnus/ecosSWtools-981021/H-host-triplet/bin. You should add this directory to your path.

  2. Extract the eCos repository using the following commands:

    cd /usr/local
    bunzip2 < ecos-1.1.tar.bz2 | tar xvf -

    On completion, the eCos repository may be found in the directory /usr/local/ecos-1.1.
