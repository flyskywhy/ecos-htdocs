Contributions

Introduction

Contributions to eCos are always welcome. These may be in the form of a new package, enhancements to an existing package or bug fixes. A contribution may be from a individual or part of a collaborative project. Information on how to contribute is contained within the eCos FAQ.

External projects

The following external projects are known to the eCos maintainers. Sources are provided as is and are not supported by Cygnus at this time.

TCP/IP stack

Tristan Gingold has ported RTEMS libnetworking to eCos. This includes a FreeBSD-derived TCP/IP stack. The sources are available for download at:

   ftp://sourceware.cygnus.com/pub/ecos/contrib/io/tcpip/rtems/

PC platform HAL

Patrick O'Grady has developed an eCos HAL for the PC platform. A GDB stub is provided, allowing the developer to download applications to the target PC via a serial port. The sources are available for download at:

   ftp://sourceware.cygnus.com/pub/ecos/contrib/hal/i386/

Synthetic Linux platform HAL

Chris Provenzano has developed the eCos i386/Linux synthetic target. This is essentially a port of the eCos HAL to a Linux process and allows the developer to run eCos applications under Linux as if they were native Linux applications. The sources are included in the eCos net distribution as of release 1.2.1.

Floating point hardware support

Sergei Organov is working on support for floating point hardware within eCos. He is targeting the PowerPC processors initially. Sources are not yet available.

NEC V850 architecture HAL

Takayuki Wakabayashi has developed an eCos HAL for the NEC V850 architecture. The port was developed using the NEC V850 simulator and has not as yet been targetted at a specific hardware platform. The port is completely unsupported and has not been tested at Cygnus. The sources will be available shortly.

Motorola MPC8xxFADS platform HAL

Kevin Hester has developed an eCos HAL for the Motorola MPC8xx Family Application Development System platform. The sources are included in the eCos net distribution as of release 1.2.1.

Samsung SNDS platform HAL

Grant Edwards is working on a port to the Samsung KS32C5000 microcontroller (ARM7TDMI core) and SNDS development hardware. Sources are not yet available (estimated availability - January 2000).
