Contributing Code

How to participate

We welcome any contribution to eCos. There are many levels at which you can participate:

  • You can participate in discussions, thus letting us know how eCos might be more useful to you.
  • You can report bugs in eCos.
  • You can send us a patch for a bug you find.
  • You can contribute a whole new piece of code, such as a device driver or a whole new package.

Legal issues: assigning copyright

If you will be submitting a large patch, or a significant body of new code, you might consider assigning the copyright of that code to Cygnus so that it can be distributed as part of the core eCos component repository.

Legal information about copyright assignment, as well as the form, can be found here .

If you do not assign copyright, we are still very interested in putting your code in the contributed area of this site.

If your contributed code is small, you can mail it to the ecos-discuss@cygnus.com list. If it is large, make it available on the net and send a pointer to ecos-discuss@cygnus.com and we will put it in the contributed area ourselves.
