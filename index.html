eCos
the Embedded Cygnus Operating System


Cygnus has developed a complete Sourceware real time operating system called eCos, the Embedded Cygnus Operating System. eCos allows fine-grained configuration of the kernel and other components, thus making it possible to write applications with a very small footprint. eCos is also very portable and allows other operating systems to be implemented with alternative configurations of the eCos kernel. The current release includes a µITRON compatibility layer.

The first full net release will occur on November 2nd, 1998 and will include complete source code for multiple processor platforms, all kernel components, HAL layers, µITRON API, C run-time, math libraries, and drivers.

We are making preliminary information available right now, including the full documentation for two initial platforms (the Matsushita MN10300 and the Toshiba TX39) and the Cygnus eCos Public License, under which eCos will be released.

There is a lot of useful information on the eCos Product Page, including FAQs and endorsements. We'll have more information about third-party software vendors, support and development services on this page on November 2nd.
