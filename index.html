eCos News

Hardware support

Hardware support has been extended to many new boards and sub-systems recently:
  • ARM development board (PID) big-endian mode
  • ARM evaluation board (AEB-1) revision C and watchdog support
  • Cirrus Logic CL-PS7111 development board
  • Cogent CMA230 CPU module (ARM7TDMI)
  • Motorola MBX860 embedded controller board
  • PCI bus control
The new hardware support packages are available via anonymous CVS only at this time. Full details of all supported hardware are provided in the supported targets table. For the very latest news on eCos, check out the NEWS file.

Sourceware CD

The eCos sources and development tools are now available on CD-ROM for those who prefer not to download them directly. The Cygnus Sourceware CD includes both the eCos 1.2.1 sources and the earlier eCos 1.1 files.

Cygnus Solutions strongly believes in the benefits of open development and, to encourage this, it is now possible to obtain the most recent versions of eCos using anonymous CVS access. This facility will provide immediate access to the latest eCos features and the rapid deployment of any necessary bug fixes.

About eCos

The embedded Cygnus operating system (eCos) is an open-source, configurable, portable, and royalty-free embedded RTOS. The system comes with everything necessary to develop eCos based applications, including tools, documentation and complete sources. All eCos runtime source code is distributed under the Cygnus eCos Public License (CEPL), a derivative of the Netscape Public License.

The following functionality is provided:

  • Hardware Abstraction Layer (HAL)
  • Real-time kernel
    • Interrupt handling
    • Exception handling
    • Choice of schedulers
    • Thread support
    • Rich set of synchronization primitives
    • Timers, counters and alarms
    • Choice of memory allocators
    • Debug and instrumentation support
  • µITRON 3.0 compatible API layered on top of the basic kernel
  • ISO C and math libraries
  • Serial device drivers
  • GDB debug support

Supported hosts

The main supported host is Windows NT on x86-based hardware. The system has also been tested on Red Hat 5.x Linux (and later releases) using x86-based hardware, and on both Windows 95 and 98, but support for these hosts should be considered beta. In addition to the tested host platforms, eCos has been hosted on Linux using PowerPC hardware and on Solaris 2.5.1.

Host software

The system comes with full support for manual configuration of the system on all host platforms. In addition Windows users get a graphical configuration tool which simplifies the process of configuring a system and validating the resulting system. This tool is not yet available for Linux users. It has, however, been used successfully with Windows NT 4.0 running as a guest operating system under VMware for Linux. It is likely that new host tools, including future versions of the graphical configuration tool, will only be available as part of the Developer Kit product.

The system comes with a full set of tools needed for embedded systems development. This includes the gcc and g++ compilers, the assembler and linker and the gdb debugger. Simulators are provided for most of the target architectures.
