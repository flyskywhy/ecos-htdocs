Chapter 5. Reporting Problems

Reporting bugs and other problems is very important: it allows Cygnus to solve your problem quickly, and improves the eCos product. The effort you make in reporting problems is appreciated.

To submit a problem report, please use the web interface. If you have a CD distribution of the eCos Developer’s Kit, you should use the address http://www.cygnus.com/ecos/ecos-pr.html; if you are using the Sourceware distribution you should use the address http://sourceware.cygnus.com/ecos/problemreport.html.
