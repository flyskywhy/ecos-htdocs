Serial device drivers

eCos provides serial device drivers for the serial ports on the MN10300 stdeval1, TX39 JMR3904, ARM PID7T, and SPARClite MB8683x evaluation boards. They provide an API (documented in the eCos Reference Manual) to control serial ports directly. The standard I/O library can be configured to use them as a transport layer.
