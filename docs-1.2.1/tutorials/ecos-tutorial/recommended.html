Recommended

  • It is strongly recommended that Windows NT users install the NT 4.0 Service Pack 3 or above before installing eCos: most testing of GNUPro, Cygwin, and eCos has been done with Service Pack 3.

  • It is also recommended that Windows NT users install Internet Explorer 4.0 or later, since this will allow the Configuration Tool’s documentation pane to be searchable.

  • A Pentium II computer and 64MB or more of RAM are recommended for best build performance.

The system has been tested only in the recommended configuration above, although other configurations are expected to work.
