Known bugs in GNUPro

Here is the list of known bugs in the GNUPro Toolkit as it relates to eCos:

  • The simulators built into GDB have a memory leak when repeatedly loading and running different binaries. After a certain number of loads, there can be a VM allocation failure.

    Work around: always re-run GDB when you wish to load a different binary.

  • When CygMon is running in command line mode, the load instruction to GDB does not function correctly.

    Workaround: use GDB instead of CygMon to load executables.
