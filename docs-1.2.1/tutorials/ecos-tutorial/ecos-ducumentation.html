eCos documentation

The eCos documentation set includes Getting Started with eCos, the eCos User’s Guide, the eCos Reference Manual, and a GNUPro Reference Manual for your specific architecture.

These are available online in HTML format at http://sourceware.cygnus.com/ecos/docs-1.2.1/ and also as printed books if you have the CD distribution of the eCos Developer’s Kit.
