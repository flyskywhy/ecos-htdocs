Chapter 22. Package: I/O sub-system

Table of Contents
Option: Debug I/O sub-system
Package: Serial device drivers

Macro name

CYGPKG_IO

Type

Boolean

Description

The eCos system is supplied with a number of different device drivers. This option enables the basic I/O system support which is the basis for all drivers.

Option: Debug I/O sub-system

Macro name

CYGDBG_IO_INIT

Type

Boolean

Description

This option enables verbose messages to be displayed on the system "diag" device during I/O system initialization.
