Option: System configuration overrides uITRON

Macro name

CYGIMP_UITRON_LOOSE_CONFORMANCE

Type

Radio

Description

Do not require the rest of the system configuration to match the needs of strict uITRON standards conformance. For example a bitmap scheduler, or timeslicing, can be used with the uITRON functions, but such an environment is not strictly conformant with the uITRON specification. Of course a system with this selected can be completely conformant; but it is up to you to configure it correctly.
