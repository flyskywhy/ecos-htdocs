Option: Bessel function limit of significance

Macro name

X_TLOSS

Type

String

Description

For the Bessel functions (j0(), j1(), jn(), y0(), y1(), yn()) this option defines the maximum absolute value of the ordinate before we assume total loss of significance. This number must be a floating-point number (e.g. contains a decimal point), and should be large.
