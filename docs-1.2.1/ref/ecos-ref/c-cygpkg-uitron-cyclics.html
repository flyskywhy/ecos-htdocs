Component: Cyclic Handlers

Macro name

CYGPKG_UITRON_CYCLICS

Type

Boolean

Description

uITRON Cyclic Handlers are used with functions named xxx_cyc(); they support timing with a periodic function callback that can be dynamically turned on or off, and resynchronized with external events.

Option: Number cyclic handlers

Macro name

CYGNUM_UITRON_CYCLICS

Type

Count

Description

The number of uITRON cyclics handlers present in the system. Valid cyclic handler numbers will range from 1 to this value.
